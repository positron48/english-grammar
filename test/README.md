# Сервис для изучения английской грамматики

Веб-сервис для интерактивного изучения курса английской грамматики на основе сгенерированных JSON файлов.

## Структура

Сервис состоит из следующих компонентов:

- **index.html** - главная страница с навигацией по разделам и главам курса
- **chapter.html** - страница главы с отображением теории и inline-квизов
- **test.html** - страница финального теста с рандомизированными вопросами

## Особенности

### Навигация по курсу
- Разделы и категории глав автоматически группируются по `section_id`
- Главы отсортированы по порядку (`order`)
- Отображение уровня сложности (A1, A2, B1, и т.д.)
- Показ примерного времени изучения

### Изучение глав
- Отображение теории в формате Markdown
- Примеры с переводами
- Ключевые моменты
- Типичные ошибки
- Inline-квизы после теоретических блоков для закрепления материала

### Финальные тесты
- Генерация тестов по стратегии отбора (`stratified_by_theory_block`)
- Рандомизация вопросов
- Проверка ответов в реальном времени
- Детальные результаты с объяснениями

## Типы вопросов

Сервис поддерживает следующие типы вопросов:

- **mcq_single** - выбор одного правильного ответа
- **mcq_multi** - выбор нескольких правильных ответов
- **fill_blank** - заполнение пропуска
- **reorder** - перестановка слов в правильном порядке
- **true_false** - верно/неверно
- **error_spotting** - исправление ошибки

## Запуск

Сервис работает как статический веб-сайт. Для запуска:

1. Откройте `index.html` в браузере
2. Или используйте локальный веб-сервер:

```bash
# Python 3
python3 -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000

# PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000/test/`

## Требования

- Современный браузер с поддержкой ES6 модулей
- Доступ к папке `../chapters/` с файлами `05-final.json`

## Структура данных

Сервис ожидает, что в папке `chapters/` находятся подпапки с именами, соответствующими `chapter_id`, и в каждой подпапке есть файл `05-final.json` со структурой главы согласно схеме `02-chapter-schema.json`.

│   ├── app.js         # Главная страница (логика)
