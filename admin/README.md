# Админ-панель для просмотра курсов грамматики

Визуальная система для просмотра и валидации всех глав и курсов.

## Запуск

### 1. Генерация индекса глав

Перед первым использованием необходимо сгенерировать индекс всех глав:

```bash
cd admin
node generate-index.js
```

Или если у вас установлен Node.js глобально:

```bash
node admin/generate-index.js
```

### 2. Открытие в браузере

Откройте файл `admin/index.html` в браузере. 

**Важно:** Для корректной работы необходимо запустить локальный веб-сервер, так как браузеры блокируют CORS запросы к локальным файлам.

#### Вариант 1: Python (из корня проекта)

```bash
python3 -m http.server 8000
```

Затем откройте http://localhost:8000/admin/

#### Вариант 2: Node.js (http-server)

```bash
npm install -g http-server
cd admin
http-server -p 8000
```

#### Вариант 3: PHP

```bash
cd admin
php -S localhost:8000
```

## Функционал

### Главная страница (index.html)

- **Список всех глав** с основной информацией
- **Поиск** по названию, ID или описанию
- **Фильтры:**
  - По уровню (A1, A2, B1, B2, C1, C2, mixed)
  - По статусу валидации (валидные, с ошибками, без валидации)
- **Статистика:**
  - Всего глав
  - Валидных глав
  - Глав с ошибками
  - Всего вопросов

### Страница главы (chapter.html)

Просмотр полной информации о главе:

1. **Обзор** - основная информация, цели обучения, статистика
2. **План (Outline)** - структура главы
3. **Теория** - все блоки теории с примерами и типичными ошибками
4. **Вопросы** - полный банк вопросов с ответами и объяснениями
5. **Квизы** - инлайн-квизы
6. **Финальная версия** - полный JSON финальной версии главы
7. **Валидация** - результаты валидации с детальной информацией

## Структура файлов

```
admin/
├── index.html              # Главная страница
├── chapter.html            # Страница просмотра главы
├── css/
│   ├── style.css          # Основные стили
│   └── chapter.css        # Стили для страницы главы
├── js/
│   ├── app.js             # Логика главной страницы
│   └── chapter.js         # Логика страницы главы
├── data/
│   └── chapters-index.json # Автоматически генерируемый индекс
├── generate-index.js      # Скрипт генерации индекса
└── README.md             # Этот файл
```

## Обновление индекса

После добавления новых глав запустите:

```bash
node admin/generate-index.js
```

Или добавьте это в автоматизацию (Makefile, скрипты генерации глав).

## Использование для валидации

Система позволяет:

- ✅ Быстро находить главы с ошибками валидации
- ✅ Просматривать все вопросы и проверять их корректность
- ✅ Проверять структуру и компоновку глав
- ✅ Видеть покрытие теории вопросами
- ✅ Анализировать результаты валидации

## Технические детали

- Чистый HTML/CSS/JavaScript (без зависимостей)
- Работает локально, не требует сервера (кроме CORS)
- Читает JSON файлы напрямую из папки `chapters/`
- Автоматически форматирует JSON для удобного просмотра
- Поддерживает Markdown в контенте теории
